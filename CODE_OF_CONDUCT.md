# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## Summary

We want fastapi-restly to be a friendly, productive place for contributors of every background and experience level. In practice that means:

- Be respectful in code review, issues, and discussions.
- Assume good faith. Disagree about ideas, not people.
- Help newcomers — everyone was new once.
- No personal attacks, no targeted harassment, no off-topic flame wars.

## Scope

Applies to all project spaces (this repository, issues, PRs, discussions) and to public spaces when an individual is representing the project.

## Reporting

If you see behavior that violates this code, contact the maintainer at **rutgerprins@gmail.com**. Reports are kept confidential. The maintainer will review and respond.

## Enforcement

The maintainer may warn, edit, or remove comments, commits, code, issues, and PRs that don't align with this Code of Conduct. Repeated or serious violations may result in a temporary or permanent ban.

The full Contributor Covenant text is available at https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
