# Security Policy

## Supported Versions

Only the latest released version of fastapi-restly receives security updates.

## Reporting a Vulnerability

If you discover a security issue in fastapi-restly, **please do not open a public GitHub issue**. Instead, email the maintainer privately:

**rutgerprins@gmail.com**

Include:

- A description of the issue
- Steps to reproduce
- The version affected
- Any suggested fix or mitigation

## Response

This is a small project maintained on a best-effort basis. You can expect:

- An initial acknowledgement within a few days.
- A fix or mitigation plan once the issue is confirmed.
- Public disclosure (with credit to the reporter, unless you prefer otherwise) once a patched release is available.

Thank you for helping keep fastapi-restly and its users safe.
