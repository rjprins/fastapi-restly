fastapi_restly.db.activate_savepoint_only_mode(make_session: async_sessionmaker | sessionmaker) → None¶

When savepoint_only_mode is enabled, any changes to the database will not be committed. This is done with “create_savepoint” mode and a wrapper on engine.connect() that begins the transaction before the Session can use it. https://docs.sqlalchemy.org/en/20/orm/session_transaction.html#session-external-transaction
